# enum

코틀린의 `when`을 사용할 때는 `enum`이 필수적이기 때문에 우선 `enum`을 알아보자. 자바의 `enum`과 유사하지만 차이가 있다.

```java
// 자바
public enum Color {
	RED, ORANGE, YELLOW, GREEN;
}
```

```kotlin
// 코틀린
enum class Color {
	RED, ORANGE, YELLOW, GREEN
}
```

자바와 다르게 선언부에 `class` 키워드가 붙는다. 프로퍼티와 메서드를 갖는 enum 클래스를 살펴보자.

```kotlin
enum class Color(var r: Int, var g: Int, var b:Int) {
	RED(255, 0, 0),
	ORANGE(255, 166, 0),
	YELLOW(255, 255, 0),
	GREEN(0, 255, 0);
	
	fun rgb() = (r * 256 + g) * 256 + b
}
```
