# 함수 활용

## 확장함수
자바, 혹은 코틀린 등으로 만들어진 라이브러리의 클래스에 메서드를 추가하고 싶은 상황이 있다.
- 자바: 상속 기능으로 확장 가능
- 코틀린: 확장함수 기능으로 확장 가능

중요한 점은, 확장함수는 오버라딩이 불가능하다.

```kotlin
fun String.double() = this + this

fun main() {
	println("do".double())
}

```

## 가변 인자 함수
```kotlin
public fun <T> setOf(vararg elements: T): Set<T>

public fun <T> listOf(vararg elements: T): List<T>
```

- varargs 키워드를 사용해서 가변 인자 수신이 가능하다.

## 중위 호출
```kotlin
infix fun String.add(postfix: String) = this + postfix

fun main() {
	println("This is very ".add("good"))
	println("This is very " add "good")
}
```

- 확장함수이면서 파라미터가 한개인 함수에 활용 가능하다.
- 주로 `mapOf("key" to "value")` 에서 많이 사용한다.

```kotlin
public infix fun <A, B> A.to(that: B): Pair<A, B> = Pair(this, that)
public fun <K, V> mapOf(vararg pairs: Pair<K, V>): Map<K, V>

println(mapOf("key" to "value", "k2" to "v2"))
println(mapOf("key".to("value"), "k2".to("v2")))
println(mapOf(Pair("key", "value"), Pair("k2", "v2")))
```
